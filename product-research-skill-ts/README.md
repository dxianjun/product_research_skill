# product-research-skill-ts

TypeScript implementation of the product research skill.

## What it does

It renders a structured research report with:

1. Research question
2. Key findings (fact vs assumption)
3. Scope
4. Options and tradeoffs
5. Recommendation
6. Risks and unknowns
7. Next experiments

## Usage

```bash
npm install
npm run build
node dist/cli.js examples/sample-input.json
```

## Test

```bash
npm test
```

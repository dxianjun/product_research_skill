# 外部数据源接入验收报告

## 1. 验收目标

完成“外部数据源接入验收”，验证从外部数据输入到告警输出的端到端链路可用。

## 2. 验收范围

1. 外部数据适配：JSON feed -> 内部产品快照模型
2. 变化检测：前后快照比对生成变化事件
3. 告警触发：按规则生成 P1/P2 告警
4. 通知分发：告警进入通知中心并分发

## 3. 验收数据

- `data_sources/amazon_feed_previous.json`
- `data_sources/amazon_feed_current.json`

关键变化样例：
1. `B012345678` 价格下降 `29.99 -> 24.49`（触发 P1）
2. `B012345678` 评分下降 `4.7 -> 4.2`（触发 P2）
3. 评论量变化 `120 -> 156`（记录变化事件）

## 4. 自动化验收用例

- 测试文件：`tests/test_external_data_integration.py`
- 用例：`test_external_feed_acceptance_flow`
- 验收断言：
1. 前后数据记录数正确（2/2）
2. 变化事件数 >= 3
3. 告警条数 = 2
4. 通知条数 = 2

## 5. 验收结果

- 结论：通过
- 说明：外部数据接入链路（接入、映射、检测、告警、通知）已跑通。

# 模块 07 测试用例：市场监控

## 功能范围

- 关键词趋势汇总
- 类目趋势汇总
- 新品/品牌动态汇总

## 测试用例清单

1. `TC-01` 关键词趋势汇总  
输入：3 个关键词点（含不同搜索量和竞争度）  
预期：关键词数量、均值、Top 关键词正确

2. `TC-02` 关键词空数据  
输入：空关键词列表  
预期：返回 0 与空列表，不抛异常

3. `TC-03` 类目趋势汇总  
输入：2 个类目各 2 个时间点  
预期：类目数量正确；趋势增量计算正确

4. `TC-04` 新品/品牌动态汇总  
输入：3 条上新数据，2 个品牌  
预期：总上新数、活跃品牌数、Top 品牌正确

## 用例合理性校验

1. 主流程覆盖关键词/类目/上新三项核心能力。
2. 边界覆盖空数据场景，确保服务稳态输出。
3. 结果字段均可直接对接看板与报告模块。
4. 与 PRD 对齐：对应“市场监控中心”Should 能力最小实现。

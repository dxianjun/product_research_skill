# 模块 06 测试用例：竞品分析看板 API

## 功能范围

- 竞品总览（总数、分层分布、品牌数）
- 变化时间线（事件序列输出）
- 我方 vs 竞品均值差距分析

## 测试用例清单

1. `TC-01` 竞品总览统计  
输入：3 个竞品（覆盖 core/secondary/watch，2 个品牌）  
预期：总数、分层统计、品牌数正确

2. `TC-02` 变化时间线输出  
输入：2 条变化事件  
预期：输出 2 条时间线；顺序号连续；字段映射正确

3. `TC-03` 差距分析计算  
输入：我方 1 个 + 竞品 2 个  
预期：价格/评分/评论数差距计算正确

4. `TC-04` 空竞品差距兜底  
输入：竞品列表为空  
预期：返回 0 差距，避免异常

## 用例合理性校验

1. 覆盖三个核心 API 能力，各有独立验证。
2. 兼顾主路径与空数据边界，避免看板空集场景崩溃。
3. 指标结果可被上层报告与页面直接复用，符合 PRD 中“看板与对比分析”要求。

# 模块 02 测试用例：竞品变化检测

## 功能范围

- ASIN 快照对比
- 字段级变化识别（price/rating/review_count/title）
- 字段变化汇总

## 测试用例清单

1. `TC-01` 无变化识别  
输入：前后快照字段完全一致  
预期：返回 0 个变化事件

2. `TC-02` 多字段变化识别  
输入：价格和评分变化  
预期：识别出 `price` 与 `rating` 两类变化

3. `TC-03` ASIN 不一致校验  
输入：前后快照 ASIN 不同  
预期：抛出 `ValueError`

4. `TC-04` 变化汇总  
输入：4 个字段均变化  
预期：每个字段汇总计数为 1

## 用例合理性校验

1. 主流程覆盖：无变化与有变化两条主路径均覆盖。
2. 业务约束覆盖：ASIN 一致性校验可防止错误数据比对。
3. 输出质量覆盖：不仅验证“识别到变化”，还验证“可统计汇总”。
4. 与 PRD 对齐：对应 PRD 中“竞品变化追踪”MVP 能力。

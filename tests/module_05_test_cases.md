# 模块 05 测试用例：日报/周报生成

## 功能范围

- 日报 Markdown 生成
- 周报 Markdown 生成
- 告警分级汇总统计（日报）
- 空数据回退文本

## 测试用例清单

1. `TC-01` 日报完整数据生成  
输入：含关键变化、2 条告警（P1/P2）、优先动作  
预期：标题正确；P1/P2 计数正确；包含优先动作区块

2. `TC-02` 日报空数据回退  
输入：关键变化/告警/动作均为空  
预期：输出 `No key changes`、`No alerts`、`No actions` 回退文本

3. `TC-03` 周报生成  
输入：策略变化、市场信号、舆情摘要、下周动作  
预期：周报标题与所有区块完整输出

## 用例合理性校验

1. 覆盖主路径：日报与周报两个核心输出都覆盖。
2. 覆盖边界：空数据场景可防止报告生成失败或出现空白区块。
3. 覆盖业务关键：日报告警分级统计是运营决策入口，单独校验计数正确性。
4. 与 PRD 对齐：对应“日报/周报自动生成、模板化输出”MVP 要求。

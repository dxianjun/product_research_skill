# 模块 08 测试用例：舆情监测

## 功能范围

- 评论主题聚类
- 情感占比统计
- 风险事件识别（负面激增、评分下滑）

## 测试用例清单

1. `TC-01` 主题聚类识别  
输入：包含质量、物流、售后语义的评论文本  
预期：对应主题计数增加

2. `TC-02` 情感占比统计  
输入：正/中/负混合评论  
预期：各情感比例计算正确

3. `TC-03` 空数据情感统计  
输入：空评论列表  
预期：全部比例为 0，不抛异常

4. `TC-04` 风险事件识别  
输入：高负面占比且低评分评论集合  
预期：识别 `negative sentiment surge` 与 `rating drop risk`

## 用例合理性校验

1. 覆盖舆情模块三类核心输出：主题、情感、风险。
2. 覆盖空数据边界，避免批处理任务中断。
3. 风险事件校验直接对应 PRD 的“差评激增、评分骤降”预警场景。
4. 输出结果可直接馈送到告警中心与报告中心。
